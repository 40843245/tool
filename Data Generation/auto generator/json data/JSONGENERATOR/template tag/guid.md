# template tag
## guid
### syntax
```
guid()
```
### intro
It generates a random guid (stands for globally unified identifer)

### examples
#### example 1
##### code
```
{
  "result": 
   "guid()"
}
```
##### output
The output may be

```
{
  "result": "171618c1-6076-44fc-aae3-76a83a498c73"
}
```
